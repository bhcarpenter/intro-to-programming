About

Introduction to Programming is a course for (primarily) Interactive Development students at The Creative Circus (http://creativecircus.edu/).

Class Website

Announcements, assignments, and discussion about the lectures are facilitated using Lore. The class website for Fall 2012 is http://lore.com/intro-to-programming
