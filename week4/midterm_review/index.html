Midterm Review

The midterm will be closed note, book, friend, and computer. Test questions can be short answer, multiple choice, fill in the blank, true/false, and matching. Expect to write code on the test.

Variables

  • Declaring and instantiating variables
  • Variable identifiers
  • Reserved words

Numbers

  • Understand the different ways to write number literals
  • parseInt/parseFloat
  • Know the difference between a float and an integer
  • Basic math operations
  • Order of operations

Strings

  • Escaping special characters
  • Concatenation

Booleans

  • Creating booleans by comparing values (<, <=, >, >=, ===, !===, etc.)
  • Logical operations (&&, ||, !)

Truthiness and Falsiness

  • Know what is considered falsey (everything else is truthy)
  • Know why this matters

undefined/null

typeof operator

  • What does it do

Conditional statements

  • If, else, else if

For/While loops

  • When and how to use them
  • What the three parts of a for loop are/when they run
  • I'll probaly have you write one loop that counts up or down to a specific number, and one loop that needs to happen over and over again until some random condition happens or the user inputs a particular string into a prompt.

Functions

  • Writing functions
  • Function parameters
  • Return statements
  • How to invoke a function

Objects

  • Creating objects
  • Accessing a value in an object
  • Setting a property of an object to a value
  • Objects in objects
  • Functions in objects
  • I may ask some basic questions about the built-in document, window, and Math objects.

Graphs / Trees

  • Root
  • Leaf
  • Parent/Child/Sibling

DOM

  • The document object
  • document.getElementById()
  • Traversal (don't memorize the property names, but be able to recognize what they do)
